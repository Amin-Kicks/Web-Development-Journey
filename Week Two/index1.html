Types of CSS

  1. Inline Styling

  2. Internal Styling

  3. External Styling

  • Inline Styling:It is applied to the tag in which the text is written.The attribute (style) is used.

  • Internal Styling: It is applied at the heading of the head tag using the "style" tag.A "dot" is used to start a class name before styling while an "#" is used for an ID.

    • NB:It is best to use a particular "ID" name for particular heading or paragraph etc but a class name may be repeated.

    ISLAM

    It is coined from the Arabic word "Salaam" which means peace i.e the literal/linguistic meaning of Islam is peace.

    In broad meaning; It is the total acceptance and submission to the will of Allah.

    Islam is built upon five pillars.

    PILLARS OF ISLAM

    1. Iman(Faith)
    2. Salah(prayer)
    3. Zakat(Alms giving)
    4. Sawm(Fasting)
    5. Hajj(Pilgrimage)

  • External Styling: It is applied by opening a css file in the same folder as the html file you want to style and link them together.

    • NB:It is the most advisable styling to use.
